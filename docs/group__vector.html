◆ vector_free()

void vector_free ( Vector *  v)

Frees all memory resources associated with a Vector.

Frees all elements in list with free element then frees all data pointed to by the data in the vector.

I.e if the vector holds Node * of:

typedef struct _Node {
int *i;
} Node;

then i will be freed. The node itself can not be. To be precise what is freed is dependent on the passed free_element function If it is null only vector->data will be freed.

This is because the data sent is copied.
